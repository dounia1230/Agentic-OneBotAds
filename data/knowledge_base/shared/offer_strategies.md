# Offer Strategies

## Purpose

This file helps the assistant move from generic promotion to a concrete commercial offer.

## Offer Building Blocks

Every offer should define:
- target buyer
- desired outcome
- core deliverable
- delivery format
- timeline
- proof or reassurance
- CTA

## Entry Offer Patterns

Good entry offers:
- consultation
- trial lesson
- starter bundle
- sample pack
- discovery audit
- first-order incentive

Use entry offers when trust is low or the purchase is unfamiliar.

## Mid-Ticket Offer Patterns

Good mid-ticket offers:
- done-for-you package
- membership
- multi-session program
- seasonal collection
- premium room or travel bundle

## Price Presentation Rules

- Explain what the buyer gets.
- Show how the package reduces friction.
- Separate price from total value narrative.
- Avoid hiding basic commercial terms.

## Urgency Rules

Use real urgency only:
- limited seasonal inventory
- fixed departure dates
- appointment capacity
- launch window
- event-driven demand

Do not invent scarcity.

## Bundle Rules

Bundles work when they:
- solve one clear use case
- reduce buying effort
- increase perceived value
- make comparison easier

## Risk Reducers

Useful risk reducers:
- transparent process
- FAQs
- sample outputs
- reviews
- flexible scheduling
- clear delivery expectations

## Upsell And Retention

After the first sale, increase value with:
- refill or replenishment logic
- premium upgrades
- recurring care or support
- seasonal launches
- cross-sell bundles
