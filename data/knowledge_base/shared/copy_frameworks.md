# Copy Frameworks

## Purpose

This file gives the assistant reusable copy structures that work across landing pages, ads, outreach, and social posts.

## Framework 1: Problem -> Outcome -> Proof -> CTA

Use when the product solves a clear pain.

Template:
- problem: what is frustrating now
- outcome: what improves
- proof: why this is credible
- CTA: what to do next

## Framework 2: Audience -> Use Case -> Benefit -> Offer

Use for niche products or services.

Template:
- audience: who this is for
- use case: when they need it
- benefit: what gets easier or better
- offer: package, price point, or next step

## Framework 3: Before -> After -> Bridge

Use for service businesses and transformation offers.

Template:
- before: current messy state
- after: desired state
- bridge: how the offer gets them there

## Framework 4: Objection-Led Copy

Use when buyers are skeptical.

Common objections to answer:
- "Is this worth the price?"
- "Will this work for my case?"
- "Can I trust this provider?"
- "How long will this take?"

## Framework 5: Offer Stack

Use when the commercial value needs to feel tangible.

Sequence:
- core deliverable
- supporting deliverables
- convenience factors
- risk reducers
- CTA

## Headline Rules

- Lead with utility before cleverness.
- Keep the first line understandable without context.
- Avoid stacking too many benefits in one headline.
- If the offer is local or category-specific, say so.

## Ad Copy Rules

- Hook fast.
- Keep one main message per creative.
- Use audience language, not internal jargon.
- Make the CTA match platform behavior.

## Landing Page Rules

- Hero section: audience, promise, proof, CTA.
- Mid-page: objections, process, proof.
- Bottom section: offer detail and low-friction CTA.

## Email Rules

- One email, one goal.
- Use one sharp promise in the subject line.
- Keep opening lines specific.
- End with a single CTA.

## Social Post Rules

- Use a strong first line.
- Break long text into short blocks.
- Teach, show, or prove something.
- End with a clear action or takeaway.
