# Marketing Principles

## Purpose

This file is a reusable strategy baseline for projects that do not yet have deep brand knowledge.

Use it when the assistant needs to:
- clarify positioning
- choose a better offer angle
- draft copy that sounds commercially useful
- avoid vague or inflated marketing language

## Core Rule

Lead with the buyer problem, then the outcome, then the mechanism, then the proof.

Do not start with abstract brand slogans unless the user explicitly asks for branding language.

## Positioning Sequence

1. Name the audience clearly.
2. Name the pain, friction, or missed opportunity.
3. Explain the offer in plain language.
4. Show why this option is more credible, faster, safer, or easier.
5. End with a specific next step.

## Message Hierarchy

Prioritize messages in this order:
- problem clarity
- practical outcome
- proof or trust signal
- offer detail
- CTA

## Good Marketing Habits

- Prefer concrete wording over slogans.
- Prefer one strong promise over five weak promises.
- Match the offer to buyer maturity.
- Keep platform copy scannable.
- Use examples, numbers, timelines, or process details when available.
- Reduce friction by answering objections early.

## Offer Framing Rules

- Show what is included.
- State who the offer is for.
- State what makes it easier to buy now.
- State any limits honestly.
- Separate main offer, bonus, and guarantee language.

## Proof Rules

Best proof types:
- real examples
- customer outcomes
- before and after comparisons
- process transparency
- founder or operator credibility
- product detail that reduces uncertainty

Weak proof types:
- empty superlatives
- generic "high quality" statements
- hype without evidence

## CTA Rules

Choose the CTA based on intent:
- low intent: learn more, see examples, explore options
- medium intent: get pricing, request details, compare packages
- high intent: book now, schedule a call, start your order

## Objection Model

Most buyers hesitate because of one of these:
- price
- trust
- timing
- fit
- complexity
- fear of wasting money

Good copy reduces those frictions directly instead of hoping the buyer ignores them.
