# Previous Ads Examples

## Purpose

This file stores reusable ad examples and message patterns in a retrieval-friendly format.

Use it to help the assistant answer with concrete patterns instead of generic ad copy.

## Collection Rules

- Save your own historical ads first.
- Save only relevant examples for your offer, audience, and funnel stage.
- Summarize examples instead of copying large blocks of competitor copy.
- Include the lesson, not just the headline.

## Entry Schema

Each example should include:
- date_collected
- platform
- advertiser_or_brand
- objective
- audience
- hook
- body_copy_summary
- cta
- landing_page_angle
- creative_format
- why_it_likely_worked
- risks_or_policy_notes
- source_link

## Example 1
