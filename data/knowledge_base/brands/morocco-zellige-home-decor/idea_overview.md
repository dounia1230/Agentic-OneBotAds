# Morocco Zellige Home Decor

## Project Summary

A home decor project centered on zellige, tiles, or Moroccan-style surfaces and accents for design-focused residential or hospitality buyers.

## Why It Fits Morocco

- Moroccan design has strong global recognition.
- Hospitality and interior projects can use place-based design language well.
- Buyers often need visual confidence and project guidance.

## Primary Audiences

- architects and interior designers
- boutique hotels and riads
- homeowners
- export buyers

## Core Offer

- tile collections
- decor objects
- sampling
- project consultation

## Messaging Angles

- handcrafted Moroccan design character
- visual impact for hospitality and home projects
- artisan detail with project support
- custom or curated design options

## Proof And Trust Signals

- finish details
- installation visuals
- size and color specs
- project references
- sample availability

## Tone

Design-led, tactile, and clear. Avoid using "artisan" without showing specifics.
